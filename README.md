# Odin Recipes

A basic recipe website built as the first project for [The Odin Project](https://www.theodinproject.com) Foundations course. 

## 🎯 Project Purpose
The goal of this project was to demonstrate mastery of core **HTML** concepts before moving on to styling and scripting. It focused on creating a multi-page website using proper element nesting and file organization.

## 🛠️ Skills Demonstrated
* **HTML Structure:** Using headings, paragraphs, unordered lists, and ordered lists.
* **Navigation:** Implementing relative links between internal pages and directory structures.
* **Git Workflow:** Initializing a repository, committing changes with descriptive messages, and pushing to a remote server.
